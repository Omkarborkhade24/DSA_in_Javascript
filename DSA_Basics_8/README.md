Merge Sort (The "Divide and Conquer" Way)Merge Sort is a professional-grade sorting algorithm used constantly in the real world. It works by dividing the array in half over and over again until you have arrays of just 1 element (which are technically sorted). Then, it merges them back together in the correct order. It is much faster ($O(n \log n)$ time).
