Hash Tables (Key-Value Pairs)A Hash Table (or Hash Map) is like a real-world dictionary. If you want to find the definition of "Algorithm", you don't read the dictionary from page one. You jump straight to the 'A' section, then 'Al', and so on.A Hash Table uses a "hash function" to convert a key (like a string) into an index, telling it exactly where the corresponding value is stored in memory. This allows for incredibly fast lookups—usually $O(1)$ time.In standard JavaScript/Node.js, standard Objects {} act somewhat like hash tables, but the built-in Map object is the purest representation of this data structure.
