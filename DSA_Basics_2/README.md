Stacks (LIFO: Last-In, First-Out)Imagine a stack of plates: you can only add a new plate to the top, and you can only remove the plate that is currently on top. This is the LIFO principle.Stacks are heavily used in managing function calls (the "call stack" in Node.js), undo mechanisms in text editors, and browser history.Key Operations:Push: Add an item to the top ($O(1)$ time).Pop: Remove the top item ($O(1)$ time).Peek: Look at the top item without removing it.
