Trees (Specifically: Binary Search Trees)Think of a tree like a family tree or a corporate organizational chart. You have a single "Root" at the top, which branches down into "Child" nodes, which branch down further until you hit the "Leaves" (nodes with no children).A Binary Search Tree (BST) is a specific type of tree with strict rules that make it incredibly fast for searching, inserting, and deleting data ($O(\log n)$ time, just like the Binary Search algorithm we looked at earlier).The BST Rules:A node can have a maximum of two children (Left and Right).The Left child must be smaller than the parent.The Right child must be greater than the parent.
