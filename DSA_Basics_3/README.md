Queues (FIFO: First-In, First-Out)
A queue is like a line of people waiting at a coffee shop. The first person to get in line is the first person to be served. This is the FIFO principle.

In Node.js, queues are incredibly important. The Event Loop itself uses a queue (the callback queue) to manage asynchronous tasks.

Key Operations:

Enqueue: Add an item to the back of the queue.

Dequeue: Remove an item from the front of the queue.
