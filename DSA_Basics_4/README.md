Linked Lists
Unlike arrays, where data is stored in contiguous memory blocks, a Linked List stores data in separate "nodes." Each node contains two things: the data itself, and a pointer (or reference) to the next node in the sequence.

Linked Lists are great when you need to frequently insert or delete items at the beginning or middle of a dataset, as you don't have to shift all the other elements (which you must do in standard arrays).
